# SnapDiff

Daily visual change monitoring for any website.

## Quick Start
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Import this repo into Vercel.
2. Add environment variables (see `.env.example`).
3. Click **Deploy**.
